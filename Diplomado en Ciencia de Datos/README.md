# Diplomado en Ciencia de Datos — Universidad del Rosario

Este repositorio contiene el material académico del **Diplomado en Ciencia de Datos de la Universidad del Rosario**.  
Aquí se encuentran los **notebooks de Jupyter** utilizados durante las clases, los cuales contienen explicaciones, ejemplos y ejercicios prácticos desarrollados a lo largo del curso.

Cada módulo incluye notebooks de contenido y, en algunos casos, versiones **`_raw`**, que corresponden a versiones base utilizadas durante las sesiones prácticas.

---

# 📦 Estructura del Curso

## 📅 Módulo 2 — Introducción a la Ciencia de Datos y Python

### Contenido

- **01_Intro_DataSciente_Python.ipynb**  
  Introducción a la Ciencia de Datos, al ecosistema de Python y a las herramientas utilizadas en el curso (Jupyter Notebooks y Google Colab).

- **02_Estructuras_de_Datos_Fundamentales.ipynb**  
  Tipos de datos primitivos en Python, operadores básicos y estructuras de datos fundamentales como listas, tuplas y diccionarios.

- **03_Estructuras_deControl_Funciones.ipynb**  
  Estructuras de control en Python (condicionales y bucles) y definición de funciones.

- **04_Ejercicios_Python_Modulo2.ipynb**  
  Taller práctico con ejercicios integradores para reforzar los conceptos vistos en el módulo.

---

## 📅 Módulo 3 — Probabilidad y Ciencia de Datos

### Contenido

- **05_DataMining_PokeAPI.ipynb**  
  Introducción a la obtención de datos desde APIs utilizando Python y exploración de datos con la PokeAPI.

- **06_Probabilidad_Ciencia_de_Datos.ipynb**  
  Fundamentos de probabilidad aplicados a la ciencia de datos.

- **07_Estadistica_Descriptiva.ipynb**  
  Estadística descriptiva: medidas de tendencia central, dispersión y análisis exploratorio de datos.

---

## 📅 Módulo 4 — Visualización de Datos

### Contenido

- **08_Visualizacion_de_datos.ipynb**  
  Introducción a la visualización de datos en Python y construcción de gráficos para análisis exploratorio.

- **09_Ejercicios_Dataviz.ipynb**  
  Taller práctico de visualización de datos y comunicación de resultados.

---

# 💻 Requisitos

Para trabajar con este material se recomienda tener instalado:

- Python 3.9+
- Jupyter Notebook o JupyterLab
- Librerías comunes de ciencia de datos (`numpy`, `pandas`, `matplotlib`, `seaborn`, etc.)

También es posible ejecutar los notebooks en **Google Colab** sin necesidad de instalación local.

---

# 📚 Uso del repositorio

1. Clonar el repositorio
2. Abrir los notebooks en **Jupyter** o **Google Colab**
3. Seguir el contenido en el orden sugerido dentro de cada módulo
